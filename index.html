  • BreadSnap, 다른 지도와 무슨 차이인가요?


    브레드스냅은 기존의 “맛있어요~” 같은 포괄적인 리뷰 대신,
    빵 자체의 속성(식감·단맛·밀도 등) 에 집중한 리뷰 플랫폼이에요.
    리뷰를 분석해 바삭, 촉촉, 쫀득 같은 키워드 태그를 자동 생성하고,
    이를 기반으로 빵 취향 맞춤형 추천을 제공합니다.
    즉, 내가 원하는 스타일의 빵을 쉽게 찾을 수 있도록 도와주는
    빵 속성 기반 지도형 리뷰 서비스입니다.

  • 이번 전시에서의 BreadSnap


    저희 팀은 이번 전시를 위해 BreadSnap의 핵심 기능 중 하나인
    빵 추천 기능을 우선적으로 구현하여, 관람객들이 직접 참여하고
    체험할 수 있는 형태의 인터랙티브 웹 서비스를 제작하였습니다.
    완성형 플랫폼이라기보다는 사용자의 취향을 기반으로 빵집을
    추천해주는 시범 버전으로, BreadSnap이 앞으로 나아가고자
    하는 방향을 미리 경험해볼 수 있도록 기획했습니다.
